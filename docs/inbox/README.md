# YAi Documentation Inbox

| Field | Value |
| --- | --- |
| Purpose | Temporary holding area for new documentation drafts that have not yet been classified |
| Audience | Maintainers and contributors |
| Status | Active |
| Last reviewed | 2026-04-27 |

Use this folder only when a document exists but its final home is still unclear.

Before a file leaves the inbox, classify it by:

- owner
- scope
- lifecycle
- audience

Do not let the inbox become a second archive.
