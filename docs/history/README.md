# YAi Documentation History

| Field | Value |
| --- | --- |
| Purpose | Shared index for documentation history, decision records, migration notes, and spec-to-surface traceability |
| Audience | Maintainers and contributors |
| Status | Active |
| Last reviewed | 2026-04-27 |

This area tracks shared documentation history: what changed in the docs structure, why it changed, and where the durable source of truth now lives.

## At a glance

- Use [Specs changelog](specs-changelog.md) for short cross-project history entries about taxonomy, naming, placement, and history-model changes.
- Use the migration records when a one-time reorganization or rewrite needs more context than a changelog entry should carry.
- Use [Surface impact map](surface-impact-map.md) when you need to connect a doc change back to the code surfaces it describes.
