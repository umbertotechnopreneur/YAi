# Support

YAi! is an open-source project maintained by Umberto Giacobbi / UmbertoGiacobbiDotBiz.

This page explains how to ask for help, when to use community channels, and when to use private commercial contact paths.

Community support is provided on a best-effort basis through repository issues and discussions, when available.

## Community Support

Use GitHub issues for:

- reproducible bugs;
- documentation problems;
- build or installation issues;
- small feature proposals;
- security-safe technical questions.

Do not post:

- secrets;
- API keys;
- tokens;
- credentials;
- private endpoints;
- customer data;
- proprietary code;
- confidential business information.

For security issues, use [SECURITY.md](SECURITY.md) instead of public issue threads.
